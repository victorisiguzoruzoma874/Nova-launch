## Deployment Steps

1. **Deploy Contract**:
   ```bash
   cd contracts/token-factory
   cargo build --target wasm32-unknown-unknown --release
   soroban contract deploy --wasm target/wasm32-unknown-unknown/release/token_factory.wasm
   ```

2. **Run Migration**:
   ```bash
   cd backend
   npx prisma migrate dev --name add_buyback_campaigns
   ```

3. **Update Environment**:
   ```env
   VITE_FACTORY_CONTRACT_ID=<new_contract_id>
   ```

4. **Deploy Backend**:
   ```bash
   cd backend
   npm run build
   npm start
   ```

5. **Deploy Frontend**:
   ```bash
   cd frontend
   npm run build
   npm run deploy
   ```
