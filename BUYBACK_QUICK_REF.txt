```
POST   /api/buyback/campaigns/:id/execute-step  # Execute step
```


### Transaction States

1. **idle**: Ready to execute
2. **pending**: Transaction in progress (shows spinner)
3. **success**: Transaction confirmed (shows tx hash link)
4. **failure**: Transaction failed (shows error + retry button)

### Error Codes

Buyback errors reuse existing error codes:
- 4: TokenNotFound (campaign/step not found)
- 14: ContractPaused (campaign not active)
- 3: InvalidParameters (all steps completed)
- 30: ChangeAlreadyExecuted (step already executed)
- 2: Unauthorized (not creator)
