### Error Code Strategy
Instead of adding 5 new error codes (which would exceed Soroban's limit), 
we reuse existing semantically appropriate error codes:

- TokenNotFound (4) → Campaign/Step not found
- ContractPaused (14) → Campaign not active
- InvalidParameters (3) → All steps completed
- ChangeAlreadyExecuted (30) → Step already executed
- Unauthorized (2) → Not campaign creator

This maintains backward compatibility and stays within platform limits.
